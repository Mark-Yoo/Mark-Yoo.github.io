Python Basic 1

파이선의 동작순서

  • 처음으로 입력된 변수를 stack frame에 쌓기 시작
  • a와 value값의 정의
  • 5번 줄로 넘어가 a와 value 값의 변경점을 입력받는다.
  • 결과적으로 변경점이 없는 a의 값은 10 그대로, 변화가 생긴 value 값은 30이 되어 stack frame으로 입력, 그 후 다시 2번 줄로 돌아가 함수를 실행한다.
  • 3번 줄에 있는 a = value 라는 식을 입력받는다. a 는 value와 같은 값인 30을 stack frame에 저장한다.
  • 값을 출력하라는 명령을 받는다. 출력값은 30이 나온다.
  • 값을 출력한 뒤 stack frame에 마지막으로 들어왔던 값은 제거되고 처음 들어간 값이 남는다.
  • 결국 다음으로 나오는 a는 10울 출력하면서 서로 다른 a 값을 출력하게 된다.
