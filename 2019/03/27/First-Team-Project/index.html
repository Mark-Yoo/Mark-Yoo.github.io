First Team Project

첫 팀 프로젝트

가제: 오늘의 영화

본 프로젝트의 목표

  • 실제 사람들이 사용하는 다이어리의 한 기능을 컴퓨터로 옮겨 더욱 쉽게 관리하고자 함
  • 다이어리의 기능 중 영화, 책 리뷰하기를 별도의 웹 앱으로 만들자는 개념에서 출발
  • 로그인 기능이 제공되지 않더라도 로컬 스토리지를 사용해 개인의 컴퓨터에 저장되게 만들고자 함

핵심 기능

  • 고전 명작 영화들을 장르별로 나열하고 각 영화의 간단한 소개와 트레일러 등을 전달
  • 이미 시청한 영화에 체크 표시 입력 기능
  • 체크 표시가 완료된 영화는 리뷰 작성이 가능
  • 작성 완료된 리뷰는 리뷰보기 창에서 확인이 가능
  • 가지고 있는 영화 리스트 중 랜덤으로 한 개의 영화를 추천하는 기능
